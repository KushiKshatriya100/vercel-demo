## Project Overview
This is a simple **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.  
Players can choose between rock, paper, or scissors, and compete against the computer, which randomly selects its choice. The game displays the winner and keeps score.

This project is perfect for beginners to practice DOM manipulation, event handling, and game logic in JavaScript.

---

## Features
- Play **Rock, Paper, Scissors** against the computer  
- **Randomized computer moves**  
- **Score tracking** for player and computer  
- Responsive and interactive **UI with CSS animations**  
- Easy to understand and extend
